# Search-Engine By TF-IDF & NLP
Download the Repo and Extract it
# Install

```shell
python -m venv env
cd /env/Scripts
activate env
back to root directory by {cd ../} one step back
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

# RUN

* Run server
```shell
flask run
```
